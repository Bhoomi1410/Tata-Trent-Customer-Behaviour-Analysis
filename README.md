# 🛍️ Tata Trent Customer Shopping Behaviour Analysis
### Zudio & Westside Retail Analytics

## 📌 Project Overview
Analysis of 5,000+ retail transactions from Tata Trent brands 
(Zudio & Westside) to uncover customer shopping patterns and 
predict repeat purchases using Machine Learning.

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Scikit-learn)
- Power BI (5 Interactive Dashboards)
- Logistic Regression Model

## 📊 Key Results
- ✅ Model Accuracy: 86.1%
- ✅ 420 loyal customers correctly identified
- ✅ 5 Power BI dashboards built
- ✅ 5,000+ transactions analyzed

## 📁 Repository Structure
- notebook/ → Google Colab Python file
- dashboards/ → Power BI .pbix file
- images/ → Dashboard screenshots & infographics
- report/ → Full project report PDF
